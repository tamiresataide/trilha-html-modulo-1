História dos computadores

Os primeiros computadores eram máquinas enormes que ocupavam salas inteiras e eram utilizados para realização de cálculos. Com o passar do tempo e o avanço da tecnologia, o tamanho desses computadores foi diminuindo, até a chegada dos personal computers, também conhecidos como pc, ou pcs.

Os computadores consistem em diversos componentes diferentes. Os hardwares são os componentes físicos, como por exemplo o mouse e o monitor. Já o software são os programas instalados na máquina. Desde um simples editor de texto, como um jogo, ambos são softwares.

O que são Clients?

Cliente, ou client na linguagem web é aquele que acessa a sua página. O usuário que abre o site e acessa as informações nele contidas, é o cliente em questão. O site realiza a comunicação entre o cliente e o servidor.

O que são Servers?

O servidor é um computador ou conjunto de computadores, dedicado ao armazenamento e distribuição de informações. Existem diferentes tipos de servidores, como por exemplo:

  • Web
  • Database
  • Email
  • Streaming

Como é uma estrutura básica de um HTML?

O HTML é uma linguagem de marcação e consiste em uma série de tags que ditam para o navegador o que é o conteúdo e como o mesmo deve ser interpretado.

A estrutura básica do HTML consiste em 3 tags principais: html, header e body.

Todo o conteúdo da página está contido entre as tags html. As informações que não são mostradas ao usuário estão contidas nas tags header. As informações que são mostradas ao usuário e o conteúdo principal da página é inserido entre as tags body.

Entendendo a diferença entre listas ordenadas e não ordenadas

Existem dois tipos de listas no html. As listas ordenadas são representadas pela tag ol que vem de ordenated list. Essas listas possuem uma sequência numérica, o que significa que cada item (li - list item) automaticamente tem um número atribuído a ele, em ordem. Já na lista não ordenada, que utiliza a tag ul, que deriva de unordered list cada item tem uma representação simbolica e não numérica.
